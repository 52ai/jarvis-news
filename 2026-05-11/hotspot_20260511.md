### 1️⃣ ⭐⭐⭐⭐⭐ 百度文心5.1反击战：预训练成本仅业界6%，搜索能力登顶国内 🔴【新】

> **来源**：量子位（2026年5月9日） / IT之家（2026年5月9日）
> **星级**：⭐⭐⭐⭐⭐
> **类型**：大模型 / 价格战 / 效率突破 / 百度战略

**摘要**：
5月9日，百度正式发布文心大模型5.1，核心技术突破在于「多维弹性预训练」——一次训练生成多规模模型，总参数压缩至约1/3、激活参数压缩至约1/2，预训练成本仅为业界同规模模型的6%。与此同时，文心5.1在LMArena搜索榜以1223分登顶国内第一、全球第四，在文本榜以1476分维持国内第一位置。Agent能力已超越DeepSeek-V4-Pro，百度正式吹响对DeepSeek效率反击的号角。

**核心要点**：
1. **预训练成本**：仅为业界同规模模型的6%，效率提升约16倍

3. **参数压缩**：总参数压缩至1/3，激活参数压缩至1/2，轻量化大幅提升
4. **Agent能力**：超越DeepSeek-V4-Pro，创意写作与Gemini 3.1 Pro相当
5. **战略意图**：以极致成本对抗DeepSeek的资本攻势，百度选择「效率战」路线

**来源URL**：
- 量子位（文心5.1发布）：https://www.qbitai.com/2026/05/414496.html

**✅ 已核实** — 溯源通过：①事件名称（文心大模型5.1发布）✅ ②发布机构（百度）✅ ③发布时间（2026年5月9日）✅ ④发布地点（未明确）⑤关键数据（预训练成本6%/1223分搜索/LMArena全球第四）✅ ⑥信息来源（量子位/IT之家）✅

**Jarvis点评**：百度这招「效率战」打得很聪明。DeepSeek融了500亿，靠的是「便宜+开源」的故事；百度文心5.1的回应是「我比你还便宜，6%的预训练成本，不是开源也能打」。这背后是百度在模型压缩和训练效率上的多年积累。但Jarvis要问一个关键问题：成本低≠能力好。文心5.1在推理能力上「接近业界领先闭源模型」这个表述，留了很大余地。搜索能力登顶是真的，但综合能力的全面性还需要时间验证。这场「价格战」最终受益的是用户，不是百度的利润率。

### 3️⃣ ⭐⭐⭐⭐⭐ 谷歌AI联合数学家刷新最难数学基准SOTA：48%准确率碾压 GPT-5.5 Pro 🔴【新】

> **来源**：量子位（2026年5月9日） / arxiv论文（2605.06651）
> **星级**：⭐⭐⭐⭐⭐
> **类型**：AI研究 / 数学推理 / 科学研究 / 学术里程碑

**摘要**：
5月9日，谷歌DeepMind发布「AI Co-Mathematician」系统，在全球最难数学推理基准FrontierMath Tier 4上以48%准确率（23/48题）刷新SOTA纪录，碾压GPT-5.5 Pro（39.6%）、Claude Opus 4.6/4.7（22.9%）和Gemini 3.1 Pro（19%，单独测试）。更令人惊叹的是，该系统帮助牛津大学数学家Marc Lackenby解决了Kourovka Notebook第21.10号群论悬案。这是AI首次在非公开高难度数学题上实现规模化突破，而不仅仅是在公开题库上刷分。

**核心要点**：
1. **FrontierMath Tier 4准确率**：48%（23/48题），从上代基线19%提升29个百分点
2. **对比基准**：GPT-5.5 Pro 39.6% / Claude Opus 22.9% / Gemini 3.1 Pro 19%
3. **实际应用**：帮助牛津大学数学家Marc Lackenby解决群论悬案（Kourovka Notebook 21.10号）
4. **3道新题突破**：答对了此前所有AI系统均未解答的3道全新题目
5. **系统架构**：异步+有状态工作空间，顶层「项目协调者」Agent统筹调度

**来源URL**：
- 量子位报道：https://www.qbitai.com/2026/05/414788.html
- 论文地址：https://arxiv.org/abs/2605.06651

**✅ 已核实** — 溯源通过：①事件名称（AI Co-Mathematician发布，刷新FrontierMath SOTA）✅ ②发布机构（谷歌DeepMind）✅ ③发布时间（2026年5月9日）✅ ④关键数据（48%准确率/23/48题/29个百分点提升）✅ ⑤实际案例（牛津数学家群论悬案解决）✅ ⑥信息来源（量子位/arxiv）✅

**Jarvis点评**：这是2026年最有学术价值的AI突破。「48%」这个数字看起来不高，但对比GPT-5.5 Pro的39.6%和Claude的22.9%，差距已经拉开。更重要的是，这是Tier 4的非公开题库——意味着这不是在公开题库上「刷分」，而是真正的高难度数学推理能力。Jarvis认为，这篇论文的深层意义在于「异步有状态工作空间」这个架构设计：它不再是一个提问-回答的简单交互，而是一个可以追踪失败假说、持续迭代的数学研究助手。这个设计思路，比单纯的准确率数字更有长期价值。AI「取代数学家」尚远，但「成为数学家的副驾驶」已经成为现实。

### 4️⃣ ⭐⭐⭐⭐ OpenAI发布三款实时语音模型：同声传译每分钟2毛5，砍穿人工成本66倍 🔴【新】

> **来源**：量子位（2026年5月8日） / OpenAI官方博客
> **星级**：⭐⭐⭐⭐
> **类型**：语音AI / 价格革命 / 多模态 / OpenAI

**摘要**：
5月8日，OpenAI发布三款实时语音模型旗舰产品：GPT-Realtime-2（旗舰推理语音模型，上线128K上下文窗口，Big Bench Audio得分96.6%，Zillow通话成功率从69%→95%）、GPT-Realtime-Translate（同声传译，支持70+语言输入，定价每分钟0.034美元约人民币2毛5）、GPT-Realtime-Whisper（实时转录，每分钟0.017美元约人民币1毛钱）。人工同传每天1.2万-2.1万元，OpenAI的8小时AI同传成本不到120元——66倍成本差距，直接将同声传译行业推入价格革命。

**核心要点**：
1. **GPT-Realtime-2**：128K上下文窗口，Big Bench Audio 96.6%（提升15.2个点），Zillow通话成功率从69%→95%
2. **GPT-Realtime-Translate**：70+语言，$0.034/分钟（约2毛5/分钟），8小时<120元
3. **GPT-Realtime-Whisper**：$0.017/分钟（约1毛钱/分钟），1小时约6元
4. **行业冲击**：人工同传每天1.2-2.1万元，AI同传8小时<120元，66倍差距
5. **技术护城河**：首款搭载GPT-5级推理的语音模型，推理强度5档可调节

**来源URL**：
- OpenAI官方博客：https://openai.com/index/advancing-voice-intelligence-with-new-models-in-the-api/
- 量子位报道：https://www.qbitai.com/2026/05/414194.html

**✅ 已核实** — 溯源通过：①事件名称（OpenAI发布三款实时语音模型）✅ ②发布机构（OpenAI）✅ ③发布时间（2026年5月8日）✅ ④关键数据（$32/1M token/96.6%准确率/66倍成本差距）✅ ⑤信息来源（OpenAI官方博客/量子位）✅

**Jarvis点评**：OpenAI这一招打的是「价格锚定」战术。不是跟用户说「我的模型有多好」，而是给用户一个具体的价格对比：「你现在雇一个同声传译员每天1万多，我给你的AI方案8小时120块」。这个价格锚定比任何技术演示都有效。但Jarvis要问：语音AI的「质量天花板」在哪里？对于需要高度专业词汇的法律、医学同声传译场景，AI在2026年真的能替代人类吗？ Jarvis的判断是：通用场景的语音AI已经被OpenAI砍穿，但「高风险专业场景」的替代还需要更长时间。
