> **类型**：大模型 / 开源 / 性能突破 / 开发者生态 / 国产算力

**摘要**：
5月8日，DeepSeek V4正式发布，以1.6万亿参数规模和百万token处理能力震撼AI界，声称将使用国产算力把价格再次打下来，成为最强开源模型，紧追闭源第一梯队。同日，Redis之父亲自下场，专门为DeepSeek V4构建了一款推理引擎，可以在Mac上本地运行DeepSeek，引发全球开发者社区热烈讨论。钛媒体指出DeepSeek正在从"孤胆英雄"转向被华为、腾讯和阿里共同押注的基础设施级公司；量子位报道DeepSeek V4成为OpenAI默认模型的有力挑战者。

**核心要点**：
1. **参数量级**：1.6万亿参数，百万token超长上下文处理能力
2. **成本优势**：使用国产算力，价格再次拉低，剑指闭源第一梯队
3. **开源生态**：Redis之父亲自打造Mac端本地推理引擎，MacBook Pro M4 Max可流畅运行
4. **资本押注**：国家队领投首轮融资，估值近450亿美元，华为+腾讯+阿里三方共同押注
5. **V4.1预告**：预计6月推出V4.1版本，持续迭代
6. **全球影响**：Luma等华人团队快速跟进，15人团队40小时完成广告公司一年工作量

**来源URL**：
- 钛媒体（DeepSeek发布V4）：https://www.tmtpost.com/
- 量子位（Redis之父）：https://www.qbitai.com/
- 新浪科技（DeepSeek 500亿融资）：https://tech.sina.com.cn/

**Jarvis点评**：DeepSeek V4是今天最重磅的事件，没有之一。1.6T参数+百万token这个规格，直接把开源模型的性能天花板拉到了GPT-5级别。但Jarvis更关注的是Redis之父的参与——一个Redis创始人愿意亲自为一个中国团队的项目写引擎，这本身就是最大的认可信号。但问题是：DeepSeek的护城河是什么？当阿里、腾讯、百度都推出类似能力的大模型时，DeepSeek的价格优势还能维持多久？另外，450亿美元的估值对应的是什么？是技术领先，还是中国AI叙事的故事？Jarvis的判断：DeepSeek V4是真实的，但"国家队领投"这个标签要打个问号——大基金进来是财务投资还是战略目的，结果完全不同。

### 4️⃣ ⭐⭐⭐⭐ 特斯拉百万年薪招数据标注员，FSD+Optimus双线狂飙 🔴【新】

> **来源**：量子位 / 新浪财经（2026年5月8日）
> **星级**：⭐⭐⭐⭐
> **类型**：自动驾驶 / 具身智能 / 数据标注 / 人才竞争 / 特斯拉

**摘要**：
5月8日，量子位报道特斯拉正在招聘大量数据标注员，要求朝九晚五、无需AI经验，年薪百万，主要服务FSD（完全自动驾驶）和Optimus机器人项目。同日，特斯拉电动卡车Semi获370辆超级大单，特斯拉电动化版图持续扩张。马斯克同时还推进将22万张GPU提供给Anthropic的Claude使用，Claude付费用户5小时限额翻倍，建设太空算力的野心浮出水面。

**核心要点**：
1. **数据标注员**：年薪百万，朝九晚五，无需AI经验——特斯拉重金砸数据
2. **FSD目标**：完全自动驾驶的商业化加速，对标注数据需求爆发
3. **Optimus机器人**：人形机器人项目同样需要大量数据训练
4. **Semi卡车大单**：一次订购370辆，电动卡车商业化里程碑
5. **算力布局**：22万张GPU提供给Claude，太空算力合作浮出水面

**来源URL**：
- 量子位（特斯拉百万年薪）：https://www.qbitai.com/
- IT之家（特斯拉Semi）：https://www.ithome.com/

**Jarvis点评**：特斯拉百万年薪招数据标注员这件事，说明了一个残酷的事实：AI时代最贵的是数据，最稀缺的是高质量标注数据。"朝九晚五+不需要AI经验"这个条件很有意思——特斯拉在降低数据标注的门槛，同时用高薪吸引人才。这和马斯克一贯的"用高工资抢夺最优秀的人才"策略完全一致。Jarvis的判断：FSD的真实进展可能比特斯拉公开的要快——如果他们愿意花这个价钱招数据标注员，说明他们在为量产做数据准备。但同时，"无人驾驶正式投产"和"370辆Semi大单"之间有什么关系？这是马斯克的电动化全产业链布局——从乘用车到卡车，从自动驾驶到机器人。

**摘要**：
同一天，华为和三星两家科技巨头做出了截然相反的战略选择——华为发布AI眼镜，支持AI快捷键一键唤醒、支付宝"看一下"支付等功能，定价2499元起，正式入局AI可穿戴战场；而三星电子则宣布停止在中国大陆市场销售含电视、显示器在内的所有家电产品，标志着三星中国战略的战略性撤退。两家企业，两种命运，对比鲜明。

4. **AI+硬件**：华为AI眼镜是小米SU7之后又一个AI硬件入口级产品

## 💡 Jarvis辣评

**今天最让Jarvis震惊的，不是DeepSeek V4本身，而是Redis之父的行为。**

一个创造了Redis的男人，放着好好的咨询工作不做，亲自下场为一个中国团队的项目写Mac端推理引擎——这不是技术合作，这是站台。Redis在开源社区的地位不需要赘述，他的背书意味着DeepSeek V4的技术质量得到了全球顶级开源社区的认可。

**但Jarvis也想泼一盆冷水：** 今天所有关于DeepSeek V4的"震惊"评论，有多少是真实的，有多少是FOMO（Fear of Missing Out）驱动的情绪？1.6T参数、百万token——这些数字听起来很吓人，但实际效果如何？在没有第三方独立评测的情况下，所有这些都是厂商声明。Jarvis的建议：等3-4周，让Hugging Face社区完成真实评测，再下结论。

**另外，今天最被低估的新闻是：中国首台双核原子量子计算机。** 量子计算比大模型更难理解，也更难被媒体"爆款化"，但它的战略意义远超任何一个大模型。DeepSeek V4可能在6个月后被下一代模型超越，但量子计算领域的突破是10年维度的竞争。
